Custom Post Builder Plugin

Home / Product / Custom Post Builder Plugin
  • Description
  • FAQs
  • Documents
  • Changelog

Associate a Divi Library Item to a Custom Post Type’s Single Page Template.

Make full use of Divi Page Builder to design your single page. Supply custom data to the Divi Template using field formatters and add_filter. Pre-baked field formatters for ACF Field are provided. Customizer settings for selecting templates for custom post types is also available.

Map a Divi Library item to a WordPress Single Page template for a public post type. The Divi Library item post content will be rendered for the post type’s single page.

We provide a Divi Module which helps you inject the dynamic data belonging to the post via a hook referred to as a field formatter. We provide set of pre-defined field formatters which are integrated with the Advanced Custom Fields (ACF) plugin.

  • Theme CustomizerPanel where you can enable and select a Divi Library item to use when rendering the single page. Each post type will have a Theme Customizer Panel. More details here.
  • A Divi Module to hook in the data for the post via field formatters for Advanced Custom Field plugin. More details here.
  • A set of pre-defined field-formatters for ACF fields.
  • Ability to define custom field formatter via hook. Details here More details here.
  • Fully Documented and increasing all the time.

How do I add and style the standard post fields like title, content etc?

We have a few modules inbuilt which can help you do that. For more visit https://www.diviframework.com/topic/post-fields-as-divi-modules/

Do you have the functionality where I can design post type single page templates?

No, we currently do not have such a functionality. Its on our radar but we cannot give any ETA on when its going to be implemented.

Can I add my own field formatter?

Absolutely. More details here.

Are there any plugin dependancies?

Yes and no. Advanced Custom Fields Pro plugin is required if you plan to use existing field formatters.
